# Product Documentation

Product and engineering documentation for AI Tech Quest.

This folder is for people who want to understand the product concept, game loop, technical architecture, API design, and implementation roadmap.

## Key Documents

- `core_concept.md`: product concept and player experience.
- `level_design.md`: mission design for RAG, ML, business assistant, and portfolio unlock.
- `tech_stack.md`: recommended stack and monorepo structure.
- `architecture.md`: frontend, backend, RAG, ML, business, and Unity integration architecture.
- `api_design.md`: FastAPI endpoint contracts.
- `development_plan.md`: phased build plan.
- `repo_structure.md`: monorepo folder ownership and future extension map.
- `unity_plan.md`: future Unity WebGL mission plan.
- `mvp_tasks.md`: MVP task breakdown.
- `codex_prompts.md`: reusable prompts for future implementation sessions.

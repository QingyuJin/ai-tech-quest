# Core Concept

## 核心設計

不要做普通作品集。  
要做一個「玩過就知道你有技術力」的互動展示。

## 使用者體驗

使用者不是被動閱讀，而是：

1. 點擊開始
2. 進入 AI Lab
3. 跟 AI Guide 對話
4. 選擇技術任務
5. 完成互動挑戰
6. 看見背後技術解說
7. 解鎖完整作品集

## 關鍵感覺

- 像小遊戲
- 像技術展覽
- 像作品集
- 像 AI Demo
- 像你自己的品牌

## 美術方向

你有電繪能力，所以要用「半手繪 + 科技 UI」做差異化。

建議風格：
- 深色實驗室背景
- 清楚 UI 卡片
- 手繪角色
- 少量動畫
- 不要過度花俏

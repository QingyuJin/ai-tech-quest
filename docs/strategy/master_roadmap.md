# Master Roadmap

## Main Direction

Build one memorable flagship project first:

**AI Tech Quest: an interactive AI technology showcase game.**

Qingyu AI Lab remains the broader brand concept, but AI Tech Quest is the main public-facing repo and demo experience.

## Why This Direction

Qingyu's strongest combination is:

- Computer science fundamentals.
- AI / ML / RAG experimentation.
- Full-stack product development.
- Unity interaction design.
- Digital art and visual UI taste.
- Codex-assisted engineering workflow.

The best showcase is not a static portfolio. It is an interactive technical product where each mission proves one skill area.

## Monorepo Structure

```text
ai-tech-quest/
├── apps/frontend-web
├── apps/backend-api
├── unity/webgl
├── docs/product
├── docs/strategy
├── assets
├── portfolio-modules
└── templates
```

## Portfolio Module Roles

1. `apps/frontend-web`: the playable interactive showcase.
2. `apps/backend-api`: mock-first API that grows into real AI services.
3. `portfolio-modules/ai-business-assistant`: freelance and monetization module.
4. `portfolio-modules/tw-civic-rag`: high-signal RAG and evaluation module.
5. `portfolio-modules/unity-ai-tutor`: Unity differentiation module.
6. `portfolio-modules/ml-experiment-lab`: technical depth module.
7. `portfolio-modules/portfolio-website`: case study and contact entry point.

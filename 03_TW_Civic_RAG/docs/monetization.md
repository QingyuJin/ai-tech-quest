# Monetization

TW Civic RAG is both a high-technical-depth portfolio product and a future document assistant service package.

## Productized Offers

### 1. Tutoring Center Document Assistant

Use cases:

- Course rules.
- Tuition policies.
- Absence and makeup class rules.
- Parent FAQ.
- Student handbook.

Value:

- Reduce repeated admin replies.
- Help staff answer consistently.
- Let parents or students find rules faster.

### 2. Small Company SOP Assistant

Use cases:

- onboarding docs
- HR rules
- internal SOP
- sales FAQ
- support docs

Value:

- Faster internal search.
- Consistent answers for staff.
- Easier onboarding.

### 3. Contractor Document Assistant

Use cases:

- project requirements
- maintenance records
- service rules
- customer notes
- quotation references

Value:

- Help admins find job-related details.
- Support future BuildFlow workflow integration.
- Reduce manual document lookup.

### 4. Club / Event Organization Document Assistant

Use cases:

- event rules
- registration guides
- volunteer manuals
- sponsor documents

Value:

- Help organizers answer common questions.
- Keep answers tied to official documents.

## Pricing Direction

Early package ideas:

- Demo setup: small document set, basic QA, citations.
- MVP setup: upload flow, vector search, simple admin UI.
- Monthly maintenance: document updates, evaluation checks, prompt tuning.
- Custom integration: LINE Bot, website chat, internal dashboard.

## Why This Can Sell

Many small teams do not need a large enterprise knowledge base. They need a practical assistant that can:

- search their existing documents
- cite sources
- avoid hallucination
- be explained simply
- deploy affordably

## Relationship To AI Tech Quest

TW Civic RAG can become:

- a Level 1 upgrade for AI Tech Quest
- a portfolio case study
- a reusable backend module for AI Business Assistant
- a paid document assistant package

## Risks

- Poor PDF extraction quality.
- Hallucinated answers if grounding is weak.
- Traditional Chinese retrieval quality varies by embedding model.
- Client documents may contain private data.
- Users may over-trust answers without checking citations.

## Risk Controls

- Citation-first UI.
- Unknown-answer behavior.
- Evaluation fixtures.
- Clear disclaimer that answers come from uploaded documents.
- Private deployment options for sensitive clients.

# Background Skills

## Product Name

## Problem

This folder keeps background skills visible without positioning them as main products.

- Interviewers who want evidence of programming fundamentals.
- Qingyu, when organizing older learning work.
- Reviewers who want to distinguish product projects from academic or practice repositories.

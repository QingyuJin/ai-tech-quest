# AI Business Assistant

Small business AI assistant project for practical freelance use cases.

## Repository

```text
https://github.com/QingyuJin/ai-business-assistant
```

## MVP Features

- One-page business website.
- FAQ management dashboard.
- Customer question and answer flow.
- Mock FAQ matching first, OpenAI API later.
- LINE / IG / Google Map entry points.
- Deployable demo.

## Recommended Tech Stack

- Frontend: React / Vite / Tailwind
- Backend: FastAPI
- Database: SQLite first, Supabase later
- AI: FAQ matching first, OpenAI API later
- Deploy: Vercel + Render

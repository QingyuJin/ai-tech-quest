# Portfolio Modules

Supporting projects that AI Tech Quest can unlock, preview, or integrate with.

These modules keep Qingyu AI Lab's broader project ecosystem visible while keeping `ai-tech-quest` as the main public-facing repo.

## Modules

```text
portfolio-modules/
├── ai-business-assistant/  Freelance-oriented AI FAQ assistant
├── tw-civic-rag/           Trustworthy Traditional Chinese document QA
├── unity-ai-tutor/         Unity + AI learning system
├── ml-experiment-lab/      Colab and ML experiment portfolio
└── portfolio-website/      Website and case study entry point
```

Each module can become its own repository later, while AI Tech Quest remains the interactive hub.

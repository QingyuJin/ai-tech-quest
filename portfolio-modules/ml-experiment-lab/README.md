## Project Direction

Each notebook should read like an engineering experiment, not just homework output.

Every experiment should include:

1. Problem
2. Dataset
3. Baseline
4. Improved Method
5. Evaluation
6. Error Analysis
7. Product Connection

- AI CUP Table Tennis Time-Series Prediction
- Netflix User Segmentation
- Data Selection Lab
- SVM / Decision Tree / AdaBoost baseline collection
- Stock time-series practice

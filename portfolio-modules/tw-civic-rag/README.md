# TW Civic RAG

Traditional Chinese trustworthy document QA system for public information, PDFs, announcements, and civic documents.

## Repository

```text
https://github.com/QingyuJin/tw-civic-rag
```

## Core Capabilities

- PDF / web document loading.
- Text chunking.
- Embedding.
- Vector search.
- RAG answer generation.
- Source citation.
- Evaluation.
- Hallucination and unknown-answer handling.

## Recommended Tech Stack

- Python
- FastAPI
- LangChain / LlamaIndex or custom pipeline
- FAISS / Chroma / Supabase Vector
- React frontend
- Colab experiments
- Vercel / Render deploy

## Completion Standard

- Upload or fixed document set.
- Question answering with citations.
- Retrieval and citation evaluation.
- Technical report.
- Demo URL and GitHub README.
